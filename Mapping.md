# Mapping

## Deck Mapping

>`DECKID` can be replaced with `Active` instead of an ID to use the active deck.

- Map Menu
  ```
   Map
  ```
- Map Deck
  ```
   Map [DECKID]
  ```
  
- Alt Map Deck
  ```
   Amap [DECKID]
  ```
  
- Full Combo Map Deck
  ```
   Fmap [DECKID]
  ```
  
- Compact Combo Map Deck
  ```
   Cmap [DECKID]
  ```
  
- Map Menu Shows
  ```
   Map Show
  ```
  
- Map Menu Stones
  ```
   Map Stones
  ```
  
- Map Menu Inventory
  ```
   Map Inventory
  ```
  
- Map Menu Clash
  ```
   Map Clash
  ```
  
- Map Menu Character
  ```
   Map Char
  ```
  
- Map Menu Object
  ```
   Map Object
  ```
  

  
## Inventory Mapping

>Filters are optional and can be combined in any order.


- Map Mythic Inventory
  ```
   Map Mythic
  ```


- Map Inventory
  ```
   MI [FUSION] [SHOW] [TYPE] [MASTERY] [RARITY] [TRAIT] [SKILL]
  ```
  
- Map Combo Mastery 

  ```
   MCM [SHOW] [MASTERY] [TRAIT] [SKILL]
  ```
  
- Map CC

  ```
   CC [SHOW] [TRAIT] [SKILL] [CHARACTER]
  ```
  
- Map CC Combo CM

  ```
   CC [NAME] CM
  ```
  
- Map PC Combo CM

  ```
   PC [NAME] CM
  ```
